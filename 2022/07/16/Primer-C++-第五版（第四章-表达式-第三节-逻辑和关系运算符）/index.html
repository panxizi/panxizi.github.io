Primer C++ 第五版（第四章 表达式 第三节 逻辑和关系运算符）

前言总结：

  1. 关系运算符
    • 作用于算术类型或指针类型
    • 返回值：布尔类型 ；值为0的表示假， 否则表示真
    • 运算对象和求值结果都是右值
  2. 逻辑运算符
    • 作用于任意能转换成布尔值的类型
    • 返回值：布尔类型 ；值为0的表示假， 否则表示真
    • 运算对象和求值结果都是右值

逻辑与和逻辑或运算符

逻辑与 （ && )

  1. 当且仅当两个运算对象都为真时结果为真
  2. 先求左侧运算对象的值再求右侧运算对象的值
  3. 短路求值（ short-circuit evaluation)。

逻辑或（ II )

  1. 只要两个运算对象中的一个为真结果就为真
  2. 先求左侧运算对象的值再求右侧运算对象的值
  3. 短路求值（ short-circuit evaluation)。
